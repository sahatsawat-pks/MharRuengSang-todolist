# Simple Calculator

A command-line calculator application that supports addition, subtraction, multiplication, and division.

## Features

- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers (with error handling for division by zero)

## Usage

Run the calculator:

```bash
python calculator.py
```

Follow the prompts to enter the operation and numbers.

## Testing

Run the tests:

```bash
pytest test_calculator.py
```

## Requirements

- Python 3.11+
